# Beta Testing Instructions

## Installation

1. Download APK từ link được cung cấp
2. Enable "Install from Unknown Sources" trên Android
3. Install APK
4. Mở app và cấp quyền cần thiết:
   - Location (Always)
   - Storage (nếu cần)

## Testing Navigation

1. Mở app
2. Chọn điểm đến
3. Bắt đầu navigation
4. Test các tính năng:
   - Route calculation
   - Voice guidance
   - Reroute
   - Offline mode

## Testing Warnings

1. Di chuyển đến gần camera
2. Kiểm tra TTS announcements
3. Kiểm tra warning banners
4. Test các loại warnings:
   - Camera
   - Speed limit
   - Danger zone
   - Railway

## Debug Mode

### Enable Debug Overlay
1. Mở Debug Screen (từ menu)
2. Toggle "Enable Live Overlay"
3. Xem FPS và metrics

### Enable Simulator Mode
1. Mở Debug Screen
2. Chọn "Navigation Simulator"
3. Load route và start simulation

## Reporting Issues

Khi gặp lỗi, vui lòng cung cấp:
1. Mô tả lỗi
2. Steps to reproduce
3. Screenshots (nếu có)
4. Logs (xem cách lấy logs bên dưới)

## Getting Logs

1. Mở Debug Screen
2. Chọn "Upload Logs"
3. Chọn file log cần gửi
4. Copy SHA256 hash
5. Gửi logs và hash cho dev team
